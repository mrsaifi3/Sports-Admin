# Sports-Admin ⚽

> Admin panel for managing sports registrations and events.

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://sports-admin-xi.vercel.app)
[![GitHub](https://img.shields.io/badge/GitHub-Repo-181717?style=for-the-badge&logo=github)](https://github.com/mrsaifi3/Sports-Admin)

## 📋 Overview

Sports-Admin is a full-stack admin panel designed to manage sports events, user registrations, and real-time updates. Built with the MERN stack.

## ✨ Features

- User authentication (JWT-based)
- Event management (Create, Read, Update, Delete)
- Dashboard with analytics and charts
- Role-based access control (Admin / User)
- Responsive UI with modern design

## 🛠 Tech Stack

| Technology | Purpose |
|-----------|---------|
| React.js | Frontend UI |
| Node.js / Express | Backend API |
| MongoDB | Database |
| JWT / bcrypt | Authentication |
| CSS / Tailwind | Styling |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/mrsaifi3/Sports-Admin.git

# Navigate to project
cd Sports-Admin

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Set up environment variables
# Create .env files (see .env.example)

# Run backend
cd ../backend
npm run dev

# Run frontend (new terminal)
cd ../frontend
npm run dev
```
